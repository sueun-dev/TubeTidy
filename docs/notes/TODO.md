# TODO: YouTube 기술/트렌드 3줄 요약 앱

## 목표/범위
- 구글 로그인 후 유튜브 계정을 연동하면 구독 채널 목록을 첫 화면에서 보여준다.
- 무료 플랜은 구독 채널(유튜버) 최대 3개만 선택 가능하게 한다.
- 선택한 채널의 기존/신규 업로드 영상에 대해 3줄 이내 요약을 제공한다.
- 사용자가 별표로 저장한 영상은 업로드 날짜 기준으로 캘린더에 아카이빙한다.
- 요금제: 무료 3채널, 10채널 $0.99/월, 50채널 $1.99/월, 무제한 $2.99/월, 라이프타임 $19.99.

## 사용자 플로우
- 온보딩: 구글 로그인 → 유튜브 OAuth 연동 → 구독 채널 불러오기 → 채널 선택(무료 3개 제한).
- 홈: 선택 채널의 최신 영상 리스트 + 3줄 요약 + 별표 저장.
- 캘린더: 날짜별 저장된 영상 확인, 필터(채널).
- 플랜 업그레이드: 채널 선택 한도 변경, 결제, 영수증.
